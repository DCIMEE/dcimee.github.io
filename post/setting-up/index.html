Setting Up

March 6, 2023

Something about how i build up this site

Prerequisites

  • git
  • hugo

I would probably say only these two are neccessary

Content

So first go into a directory where you want to put your site files at then run the command

hugo new site "sitename"

and, clone whatever theme you like to the directory called theme (You could find out on hugo’s website or somewhere else on the internet.)

Edit the file called config.toml, i just simply copy the Sample Configuration. from the theme’s wiki.

then, run

hugo

under your siet’s root directory to genarate your site, the files would be at ./public directory (don’t forget to run this command everytime after you make changes or add new contents.)

Also I have put this site on github page, you can see it by the domain. so what i did is create a repository with “xxx.github.io” as it’s name, it seems only by doing that can you use github page feature. (you may also need to enable this feature on repo’s setting.)

then run

git init

under the public directory since thoes files in it are all you need.

run

hugo server

to check the website changes in realtime, i do edit on WSL, the default server address it given just not works for me, so what i run is

hugo server --bind WSLIP --baseURL=http://WSLIP --noHTTPCache

(You can get the WSLIP by using the command below)

ip a

Next you just need to do git push, which i won’t talk about here, so that’s all.
